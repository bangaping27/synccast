Installation Guide

How to Install SyncCast

01. Download

Click the "Add to Chrome" button above to download the synccast.zip package to your computer.

02. Extract

Locate the downloaded file and Unzip/Extract it into a permanent folder (e.g., in your Documents).

03. Developer Mode

Open Chrome and go to chrome://extensions. Switch on "Developer mode" in the top right corner.

04. Load Extension

Click "Load unpacked" and select the folder where you extracted the files. Boom! You're ready.

⚠️

Note: Since this is a custom community-built tool, keep Developer Mode ON to ensure the extension functions correctly.
